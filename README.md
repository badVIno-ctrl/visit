# badVIno — сайт-визитка

Чистая статика: HTML + CSS + JS. Никаких сборщиков и зависимостей.

## Запуск (Windows)

```powershell
# из папки visit
python -m http.server 5500
```
Открой `http://localhost:5500`.

## Структура

```
visit/
  index.html   — разметка
  styles.css   — стили (минимализм: чёрный/белый/серый + алый акцент)
  script.js    — карточки проектов, контакты, фон-канвас
  README.md    — этот файл
```

Список проектов и цены лежат прямо в `script.js` (массив `PROJECTS`) и в
`index.html` (секция `services`). Менять — там же.


## Что делает страница

- Hero с описанием, статусом «принимаю заказы», тикером языков.
- Блок «обо мне» — три колонки: что делаю, стек, как работаю.
- Сетка карточек проектов. Превью — `iframe` уменьшенный до ×0.4 +
  декоративная «шапка браузера». Если сайт-источник запрещает iframe,
  под ним остаётся стилизованная заглушка-мокап (SVG в `MOCKS`). Клик по
  карточке открывает проект в новой вкладке.
- Услуги и цены — кнопка «обсудить» сразу открывает Telegram с
  заполненным сообщением.
- Контакты раскрываются по клику (никаких прямых ссылок в открытую).
- Фон — `<canvas>` с дрейфующим алым свечением и сеткой точек, реагирует
  на курсор. Минималистично, без «стекла».


## Доступность и моб. версия

- Семантическая разметка, `aria-label` где нужно.
- `prefers-reduced-motion` уважается — анимации фона отключаются.
- Адаптив до 360 px ширины.
